## Just follow these steps to setup
1. Download python3.9+
2. Download and install Ollama
3. Install a model of your choice (llama3.2, phi2, llama3. mistral). Know your environment before you download the model

```bash
ollama pull llama3.2
```
4. Clone the repository from GitHub, open it in an IDE
5. ctrl + shift + F for global search, by default the game uses llama3, replace it with the model you downloaded using ollama.
6. Then run
```bash
python3 main.py
```
7. And done!!
